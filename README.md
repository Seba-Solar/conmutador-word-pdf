El proyecto transforma los documentos de formato Word a formato PDF

**REQUISITOS:
- Tener Python instalado
- Tener Win32com instalado**

- Crear una carpeta con el nombre de Word dentro del repositorio y meter dentro los archivos en formato Word que se desean transformar a PDF
- Crear una carpeta con el nombre de PDF dentro del repositorio, aqui se guardaran los Word que fueron transformados a PDF

Este proyecto no elimina los Words que se transformaron a PDF por lo cual si se desea transformar nuevos Words se debera quitar los ya que estan dentro de la carpeta y agregar los nuevos a transformar
